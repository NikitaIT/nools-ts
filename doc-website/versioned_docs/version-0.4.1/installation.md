## By Package Manager

```bash npm2yarn
npm install nools-ts
```

## Dev

Clone source from Github, or download source code.

```bash
$ git clone git@github.com:nikitait/nools-ts.git
```

Switch to branch functional

```bash
$ git checkout functional
```

Build

```bash npm2yarn
$ npm install
```
